# Screenshots Directory

Place your application screenshots in this directory. The following screenshots are expected:

## Required Screenshots

1. **dashboard.png** - Screenshot of the main dashboard showing the list of tracked offers
2. **offer-details.png** - Screenshot of an offer details page with price history chart
3. **add-offer.png** - Screenshot of the form for adding a new offer
4. **price-chart.png** - Screenshot showing the interactive price chart visualization

## Image Guidelines

- **Format**: PNG or JPG
- **Aspect Ratio**: 16:9 (recommended for best display)
- **Resolution**: Minimum 1280x720, recommended 1920x1080 or higher
- **File Size**: Keep under 500KB per image for optimal loading

## How to Add Screenshots

1. Take screenshots of your application
2. Save them with the exact filenames listed above
3. Place them in this directory (`/src/assets/screenshots/`)
4. The build process will bundle and optimize them automatically, and the images will appear on the landing page

The landing page will show placeholders until these images are added.
